Reconnaissance Faciale Avec IA - cloud Firebase

 Github Link

Implémenter un système capable d'identifier et de vérifier des individus en analysant leurs caractéristiques faciales (forme des yeux, du nez, de la bouche) à partir d'images.Et renforcer la sécurité des accès aux installations, pour réduire les risques de fraudes et d'automatiser la surveillance.

Technologies Utilisés : Vonage, OpenCV, Face Recognition, Python, Numpy, Os, CSV

Détection d'anomalies sur les transactions Retail en ligne

 Github Linkd

Identification les jours présentant des comportements anormales dans les ventes en ligne en s'appuyant sur le jeu de données Online Retail Transactions. En utilisant et en comparant plusieurs méthodes de détection pour anticiper des comportements inhabituels dans les transactions.

Technologies Utilisés : Scikit-learn, Matplotlib, Pandas, Python, Numpy, Seaborn
Points principales:
- EDA (Exploratory Data Analysis): Pour analyser les tendances et les saisonnalités afin de mieux comprendre la structure des données avant le traitement;
- Application des méthodes de détection: Le projet met en œuvre quatre techniques spécifiques pour isoler les anomalies: Robust Z-Score, Isolation Forest, Local Outlier Factor, DBSCAN ;
- Visualisation des résultats: Représentation sous forme de courbes temporelles, pour faciliter l'interprétation des données et la comparaison entre les différentes méthodes.
